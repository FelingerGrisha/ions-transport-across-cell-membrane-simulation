# Численное моделирование транспорта ионов через клеточную мембрану эритроцита

Данный проект представляет собой математическую и программную модель биофизических процессов, происходящих в эритроците. Программа симулирует динамику клеточного объема, мембранного потенциала и внутриклеточных концентраций основных ионов (Na⁺, K⁺, Cl⁻, Ca²⁺) с течением времени. 

Проект разработан с фокусом на алгоритмическую сложность, отказоустойчивость вычислений и чистоту кода.

## Ключевые инженерные решения (Features)

* **Кастомный математический движок:** Интегрирование системы нелинейных обыкновенных дифференциальных уравнений (ОДУ) выполняется с помощью самостоятельно реализованного **метода Рунге-Кутты 4-го порядка** (RK4) без использования высокоуровневых решателей (например, из `scipy`).
* **Отказоустойчивость (Fault Tolerance & State Recovery):** Симуляция рассчитана на миллионы итераций и длительное время работы. Внедрен механизм сохранения промежуточных состояний (через сериализацию `pickle`). Программа умеет перехватывать прерывание пользователем (`KeyboardInterrupt`), безопасно сохранять дамп памяти в файл `intermediateState.pkl` и бесшовно возобновлять вычисления при следующем запуске без потери прогресса.
* **Отделение данных от логики:** Все физические константы, стартовые параметры клетки и кинетические коэффициенты вынесены в отдельный конфигурационный файл `data.json`.
* **Строгая типизация и инкапсуляция:** Код написан с соблюдением современных стандартов Python (PEP-8, Type Hints). Исключено использование глобальных изменяемых состояний при интеграции, логика защищена конструкцией `if __name__ == "__main__":`.
* **Оптимизация:** Применена векторизация математических операций с помощью библиотеки `NumPy` для ускорения расчета производных на каждом шаге времени.

## Структура проекта

* `main.py` — основной скрипт. Содержит реализацию метода RK4, расчет производных (скоростей изменения параметров) и логику Checkpointing-а.
* `pump.py` — функция для расчета чистого потока Na⁺/K⁺-АТФазы на основе сложного шестистадийного кинетического цикла.
* `init_erith0.py` — модуль инициализации. Загружает параметры клетки, рассчитывает равновесное состояние, начальный мембранный потенциал и эффективный заряд органических анионов.
* `init_pump.py` — парсинг кинетических коэффициентов и концентраций метаболитов (ATP, ADP, P_i) для модели насоса.
* `data.json` — конфигурационный файл с начальными условиями и физическими константами.
* `requirements.txt` — список сторонних зависимостей.

## Установка и запуск

**1. Клонирование репозитория:**
```bash
git clone https://github.com/FelingerGrisha/fluid-ions-across-cell-membrane-simulation.git
cd fluid-ions-across-cell-membrane-simulation
```

**2. Установка зависимостей:**
Для работы проекта требуются `numpy` и `matplotlib`.
```bash
pip install -r requirements.txt
```

**3. Запуск симуляции:**
```bash
python main.py
```

## Использование и результаты

Во время работы программа будет циклично выводить промежуточные данные в локальные файлы `conc.dat`, `flux.dat` и `ion.dat` (добавлены в `.gitignore`).
